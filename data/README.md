# Data availability

The datasets provided in this directory are shared under the
Creative Commons Attribution 4.0 International License (CC BY 4.0).

These data support the findings of the study:
"IMMUNE SIGNATURES IN HIV INFECTION: sIL-6 AND VIRAL LOAD PREDICT ADVANCED IMMUNODEFICIENCY, WHILE ANTIRETROVIRAL THERAPY RAPIDLY REDUCES sIL-2R LEVELS"

If you use these data, please cite the original publication and this repository.

Some variables may have been anonymized to preserve participant confidentiality.
